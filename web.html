The Solution
In this project, I developed a responsive mobile site, which was later expanded into a fully functional website. The design included the use of two carefully selected fonts, color combinations, and an aesthetically pleasing grid layout. The project focused on a mobile-first approach, ensuring an optimal user experience across devices. This project enhanced both my technical and design skills and strengthened my understanding of how to create modern, responsive web design.

Code

I worked with both grid and flexbox to create a flexible and scalable layout and used media queries to ensure the design adapted to different screen sizes. Additionally, an intuitive navigation menu and metadata optimization for images were implemented.

Design

On the design front, I integrated Gestalt principles, layout principles, typography, and color choices, resulting in a harmonious and user-friendly experience. I also conducted an analysis of existing websites to understand how to balance aesthetics and functionality.
